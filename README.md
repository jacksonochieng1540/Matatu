# 🚌 Matatu Booking & Scheduling System

A Django-based web application for booking seats on matatu trips in Kenya. Passengers can sign up, view available trips, and book seats, while admins/operators can manage matatus and schedules.

---

## ✨ Features

- ✅ User registration & login (with email and password)
- ✅ Admin/operator interface for adding matatus and trips
- ✅ View all upcoming trips
- ✅ Book available seats
- ✅ Prevent duplicate or overbooked seat reservations
- ✅ Responsive UI using Tailwind CSS

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, Tailwind CSS, JavaScript
- **Database:** SQLite (default) – easy to switch to PostgreSQL
- **Authentication:** Django’s built-in auth system
